# edX GraphQL API

edX is an online learning platform offering MOOCs from universities and institutions. The API covers course catalog, enrollment management, course grades, certificates, user profiles, learner activity, and content block metadata.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from edX REST API
**Documentation:** https://docs.openedx.org/

- Reference: https://docs.openedx.org/en/latest/developers/references/index.html
- Schema: https://github.com/api-evangelist/edx
